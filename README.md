# playwright-az-cli
**Custom Docker image with Microsoft Playwright and Azure CLI pre-installed**

This repository provides a custom Docker image based on the official Microsoft Playwright image, extended with Azure CLI support. The image is designed for teams and CI/CD pipelines that require browser-based automation (using Playwright) alongside Azure operations such as authentication, deployments, and resource management.

Key features:

Based on the official Microsoft Playwright image

Azure CLI pre-installed and ready to use

Suitable for CI/CD pipelines, test automation, and cloud-integrated workflows

Eliminates the need to manage separate containers for Playwright and Azure CLI

This image is ideal for scenarios where end-to-end testing, automation, or scripting workflows need tight integration with Azure services.
